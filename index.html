  • 2048 Game

    "A number-matching puzzle game with animations."

    About

    I was searching for some project ideas on the internet and came across 2048 which I am very familiar about, I was fascinated by how the tiles move and merge I realized its not an easy algorithm to do so I set myself up a challenge and decided to do it my own way. There's a lot of problems along the way but the trickiest one is to animate the tile's movement, to do this I have to get the initial and final locations of the tiles which is not that easy. My final solution was to retrieve these values from the internal function that moves the actual tiles in the array which I doubt the best solution. It's working but I'm planning to make this project better in the future.

  • Simple Chess GUI

    "A fully-functional chess interface with animations."

    About

    Chess is my favorite game, so I decide to create a chess interface and to make it challenging I did not search for any logic and code references and decided to do it my own way. The greatest problem was to make sure that pieces cannot move when pinned to the king, it took me a couple of renditions just to solve this problem. My final solution is to perform a mock move, that is to move the piece temporarily and if the king is in check then the piece is immovable. It was not perfect yet but I was able to make it fully-functional without using any chess library. I am planning to migrate it to canvas and add more funtionalities to it in the future.

    Browse Simple Chess GUI View Repository on Github
